# MobiShop - Hệ thống bán lẻ điện thoại

Chào mừng bạn đến với MobiShop! Đây là phiên bản website đã được tối ưu hóa để chạy trực tiếp trên **VS Code** mà không cần cài đặt thêm bất kỳ thư viện nào.

## Cấu trúc 3 file sạch sẽ:
1. **index.html**: Chứa khung xương và cấu trúc trang web.
2. **style.css**: Chứa định dạng giao diện, màu sắc và hiệu ứng.
3. **script.js**: Chứa toàn bộ logic xử lý (Giỏ hàng, Đăng nhập Admin, Quản lý sản phẩm).

## Cách chạy trên VS Code:
1. Mở thư mục dự án bằng **VS Code**.
2. Chuột phải vào file `index.html`.
3. Chọn **"Open with Live Server"** (nếu bạn đã cài extension Live Server) hoặc đơn giản là kéo thả file `index.html` vào trình duyệt web của bạn.

## Các tính năng chính:
- **Trang chủ**: Hiển thị danh sách điện thoại hiện đại.
- **Giỏ hàng**: Thêm, xóa, cập nhật số lượng sản phẩm.
- **Thanh toán**: Quy trình đặt hàng hoàn chỉnh.
- **Admin Dashboard**:
    - Đăng nhập: `admin` / `123456`.
    - Quản lý sản phẩm (Thêm/Xóa).
    - Quản lý đơn hàng (Cập nhật trạng thái).
- **Hệ thống Toast**: Thông báo đẹp mắt thay thế cho alert/confirm.
- **Dữ liệu**: Tự động lưu vào `localStorage` trên máy tính của bạn.

Chúc bạn có trải nghiệm tốt nhất với MobiShop!
